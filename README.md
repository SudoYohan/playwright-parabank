# Playwright Daily Exercises

This folder contains a starter Playwright framework for daily exercise testing.

## Setup
1. Open a terminal in `Playwright - Daily Exercises`
2. Run `npm install`
3. Update `playwright.config.ts` `baseURL` to the website you want to test

## Structure
- `tests/` - Playwright test files
- `pages/` - Page Object Model classes
- `playwright.config.ts` - Playwright configuration
- `tsconfig.json` - TypeScript settings

## Run tests
- `npm test`
- `npm run test:headed`
- `npm run test:report`
